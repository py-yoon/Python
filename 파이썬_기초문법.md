파이썬 기초 문법

### 입력
`input()`

#### 타입
```python
int() # 정수형
float() # 실수형
complex() # 복소수형
str() # 문자형
```

#### 타입 확인하기
입력한 값이 무슨 타입인지 궁금하다면 `type()`을 이용하자.

> 예시
```python
>>> a=3+7j
>>> type(a)
<class 'complex'>
```

### 연산
컴퓨터는 입력, 연산, 출력의 기능을 한다. 가장 기본적인 연산자는 이렇다.

```python
+ 더하기
- 빼기
/ 나누기
* 곱하기
** 승수
// 몫 구하기
% 나머지 구하기
```

> 예시
```python
>>> 2+3
5

>>> 2-3
-1

>>> 11//7
1

>>> 3**3
27

>>> 11%7
4
```

### 출력
`print()`

#### Hello World! 출력
```python
>>> print("Hello World!")
Hello World!
```
